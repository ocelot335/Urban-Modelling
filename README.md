Дополнение для macOS:
1)Скачайте JavaFX:https://openjfx.io/  
2)Используйте brew install ffmpeg  
3)Запускайте через:  java --module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml -jar GeoAut.jar
